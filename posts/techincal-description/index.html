Technische Beschreibung

February 26, 2019

Das Host You Own Minecraft Server Projekt besteht im wesentlichen aus drei GitRepositories, welche alle bei GitHub zu finden sind. Ziel ist es die Komplette Infrastruktur im SourceCode zu verwalten und öffentlich auf GitHub für die Wiederverwendung zu presentieren.

Der Blog

Star Watch Fork Issue

Der mit Hilfe von Hugo generierte Blog (nolte/minecraft-gameserver) trennt den Redaktionellen Teil des Projektes vom Technischen-Aufbau.

Die Infrastruktur

Star Watch Fork Issue

Die Infrastruktur wird aktuell bei hetzner.de betrieben und mit einer Kombination von Ansible und Terraform bereitgestellt, weitere Informationen sind im nolte/minecraft-infrastructure Repository zu finden.

Minecraft Ansible Role

Star Watch Fork Issue

Der Minecraft Server wird mit Hilfe eine Ansible Role (nolte/ansible-minecraft) konfiguriert und installiert. Diese Role ist durch Ansible Galaxy einfach nutzbar.
